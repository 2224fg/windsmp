WindSMP - strona GitHub Pages

Jak wrzucić:
1. Rozpakuj ZIP.
2. Na GitHub utwórz repozytorium np. windsmp.
3. Wrzuć pliki: index.html, style.css, script.js.
4. Settings -> Pages -> Deploy from branch -> main -> /root.
5. Poczekaj chwilę i dostaniesz link.

Co podmienić:
- W index.html zmień https://TWOJ-SKLEP.tebex.io/ na linki do Tebex.
- W index.html zmień https://discord.gg/TWOJ-LINK na Discord.
- W script.js zmień play.windsmp.pl na swoje IP.
